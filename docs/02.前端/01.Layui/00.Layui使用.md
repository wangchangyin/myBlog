# Layui使用

## 1.弹框使用

```html
layer.msg("请填写问题内容！！！",{icon: 1});//成功
layer.msg("请填写问题内容！！！",{icon: 2});//失败
layer.msg("请填写问题内容！！！",{icon: 2});//警告
layer.msg("请填写问题内容！！！",{icon: 2});//加锁表情
layer.msg("请填写问题内容！！！",{icon: 2});//哭的脸
```
